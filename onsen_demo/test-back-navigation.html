🧪 Test Back Navigation Fix

🎯 Fix Applied: Deferred Content Loading

Problem: Inline script tried to load content before functions/elements existed

Solution: Store pending content load in sessionStorage, process it after everything loads

🔧 New Flow

1. Back button clicked → sessionStorage.setItem('targetTab', 'start') 2. Navigate to index.html 3. Inline script runs: - Sets start-tab as active (prevents flash) - Sets sessionStorage.setItem('pendingContentLoad', 'start') 4. Document loads completely 5. ons.ready() or DOMContentLoaded fires 6. processPendingContentLoad() runs: - Reads pendingContentLoad - Calls loadStartWorkoutPlans(startPage) - Content appears!

🧪 Manual Test Steps

  2. Go to Start tab
